# Reactive Network Hackathon Projects

这个仓库现在按项目拆分为三个独立目录：

## 1. `reactive-event-to-callback-demo`

官方 `basic` demo 对应的最小 Reactive Network 闭环示例。

- 项目目录：`reactive-event-to-callback-demo/`
- 说明文档：`reactive-event-to-callback-demo/README.md`
- 作业提交信息：`reactive-event-to-callback-demo/SUBMISSION_BRIEF.md`

- 跑通证明：`uniswap-v2-stop-order-sepolia-demo/demo跑通证明.md`
- 技术说明：`uniswap-v2-stop-order-sepolia-demo/src/demos/uniswap-v2-stop-order/README.md`
